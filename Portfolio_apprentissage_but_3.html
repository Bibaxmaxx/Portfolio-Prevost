1. Réaliser : Adapter des applications sur un ensemble de supports

Contexte et Enjeux

Dans le domaine du développement logiciel moderne, la capacité à adapter des applications sur différents supports est cruciale. Mon parcours BUT 3 m'a permis de développer une expertise technique dans ce domaine.

Compétences Développées

  • Développement d'applications web responsive avec Vue.js
  • Création d'API REST multi-supports avec Symfony
  • Gestion des contraintes de compatibilité multi-plateforme

Réalisations Concrètes

Projet API REST - Gestion d'Événements - Développement d'une API flexible utilisable sur multiple supports - Implémentation de l'authentification JWT - Conception d'une architecture modulaire et extensible

Défis Techniques Relevés

- Gestion des différences de rendu entre navigateurs - Optimisation des performances sur mobile et desktop - Mise en place de stratégies de responsive design

Outils et Technologies

  • Vue.js 3 avec Typescript
  • Symfony et API Platform
  • Responsive CSS (Flexbox, Grid)
  • Axios pour les requêtes multi-supports
