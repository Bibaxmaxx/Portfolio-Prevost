Portfolio d'Apprentissage BUT 3

1. Compétences de Niveau 3 développées

A. Compétence : Choisir et implémenter les architectures adaptées

Durant mon parcours de BUT 3, j'ai approfondi ma capacité à concevoir des architectures logicielles robustes : - Développement d'une API REST avec Symfony et API Platform - Mise en place d'une architecture front-end modulaire avec Vue.js - Conception de systèmes découplés avec une séparation claire des responsabilités

Preuves : - Projet d'API REST de gestion d'événements - Implémentation de l'architecture MVC dans les projets web - Utilisation de design patterns comme Repository et Dependency Injection

B. Compétence : Organisation et veille numérique

J'ai développé ma capacité à gérer l'information et à rester à jour technologiquement : - Mise en place d'une veille technologique sur les frameworks web modernes - Utilisation de GitHub pour le versionning et la collaboration - Participation à des forums et communautés de développeurs

Preuves : - Dépôts GitHub documentés - Utilisation de flux RSS et de plateformes comme Dev.to - Intégration continue et gestion de versions

C. Compétence : Optimisation et performances

J'ai approfondi mes compétences en analyse et optimisation de code : - Profilage des performances avec des outils comme Blackfire - Optimisation des requêtes SQL et des algorithmes - Mise en place de stratégies de caching

Preuves : - Analyse de performance de l'API REST - Réduction des temps de chargement dans les applications web - Utilisation de techniques d'optimisation comme la lazy loading

2. Réflexion sur mon parcours

Mon parcours en BUT 3 m'a permis de passer d'un développeur junior à un professionnel plus mature. J'ai appris à : - Penser de manière systémique - Anticiper les défis techniques - Communiquer efficacement en équipe

Les principaux défis que j'ai surmontés : - Gérer la complexité des architectures distribuées - Comprendre les enjeux de sécurité et de performance - Adopter une approche centrée utilisateur dans mes développements
