Projet Front-end Vue.js - Gestion d'Événements



Description :

Développement d'une application front-end en Vue.js pour une API REST de gestion d'événements. Le projet permet aux utilisateurs de s'inscrire, consulter et interagir avec différents événements.



Rôle :

En tant que développeur front-end, j'ai été responsable de la conception et de l'implémentation de l'interface utilisateur, de la gestion des états, et de l'intégration avec l'API REST.



Technologies Utilisées :


  • • Vue.js 3
  • • TypeScript
  • • Pinia (Gestion d'état)
  • • Vue Router
  • • Axios


Fonctionnalités Principales :


  • • Authentification et gestion de compte utilisateur
  • • Consultation et inscription aux événements
  • • Gestion des erreurs et notifications
  • • Intégration avec l'API REST Symfony
  • • Utilisation de l'API Annuaire
