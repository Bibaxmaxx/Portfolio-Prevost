Projet Web Vue.js

Application de Gestion d'Événements

Projet Front-end Vue.js - Gestion d'Événements



Contexte du Projet :

Développement d'une application front-end moderne pour une plateforme de gestion d'événements, en parfaite synergie avec une API REST développée en Symfony.



Objectifs Principaux :

  • Créer une interface utilisateur dynamique et réactive
  • Intégrer une authentification sécurisée
  • Gérer les interactions avec l'API REST
  • Offrir une expérience utilisateur fluide et intuitive



Technologies et Outils Utilisés :

  • Frontend : Vue.js 3, TypeScript
  • State Management : Pinia
  • Routing : Vue Router
  • Requêtes API : Axios
  • Authentification : JWT
  • Style : Tailwind CSS



Fonctionnalités Développées :

  • Système d'authentification complet
  • Consultation et inscription aux événements
  • Gestion du profil utilisateur
  • Système de notifications dynamiques
  • Intégration avec l'API Annuaire



Défis Techniques :

  • Gestion des états complexes avec Pinia
  • Mise en place d'une authentification sécurisée
  • Optimisation des performances
  • Gestion des erreurs et cas limites



Approche de Développement :

  • Développement itératif
  • Tests unitaires et d'intégration
  • Utilisation de la méthodologie Agile
  • Revues de code régulières
